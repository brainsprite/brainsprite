Installation

Python

The Brainsprite python library is available through pypi:

pip install brainsprite

Javascript

The brainsprite javascript library is available through npm:

npm install brainsprite.js

You can also directly incorporate brainsprite.min.js as an asset of your webpage. Its only dependency is jquery. Download links are available on the brainsprite github repository.
